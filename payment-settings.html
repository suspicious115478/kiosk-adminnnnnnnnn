How to get your API Keys
  1. 1
    Create a Razorpay account Go to razorpay.com and create a free account
  2. 2
    Enable Test Mode Use the toggle in the top-right of the dashboard to select Test Mode
  3. 3
    Settings → API Keys Click Settings in the left sidebar, then go to the API Keys section
  4. 4
    Click "Generate Test Key" You will receive a Key ID and Key Secret — the secret is shown only once, so copy it
  5. 5
    For UPI ID In test mode, use success@razorpay. In live mode, go to Settings → Payment Methods → UPI
